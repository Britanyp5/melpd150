Homework 9 Data Visualization

Charts:

For this assignment, I was able to learn how to create charts in Adobe Illustrator and understand how data visualization can make information easier to understand. At first, the process was a little challenging because I was not able to quickly find the tools I needed to make the charts or figure out how to change the colors and styles. However, after exploring the program more and using the search tool, it became much easier to navigate through Illustrator and find the features I needed. The search tool was extremely helpful because it saved time and helped me learn where different functions were located within the program. While creating the charts, I learned that data visualization is not only about presenting information, but also about making the information visually appealing and easier for an audience to understand. Creating the actual pie chart was one of the most enjoyable parts of the assignment because I was able to customize it however I wanted. I experimented with different colors, layouts, and chart styles to make the information stand out more. Making the donut-style 3D pie chart was especially fun because it looked more artistic and modern compared to a basic flat chart. For example, the chart showing the “spicy chips that people like” looked more visually interesting and creative than the simpler chart displaying the “people that like spicy chips.” Instead of only looking at numbers, charts allow people to quickly understand comparisons and percentages in a visual way. I realized that infographics and data visualizations are useful because they can present research, surveys, and studies in a format that is both informative and visually attractive.
